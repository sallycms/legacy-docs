Handbuch
========

Das Handbuch führt in die inhaltliche Verwaltung einer Website auf Basis von
SallyCMS ein. Die Website wird dabei durch das CMS "live" erzeugt. Das CMS
ermöglicht die Trennung der Gestaltung und Programmierung von den Inhalten der
Website. Somit müssen Redakteure keine Programmierkenntnisse besitzen, um einen
großen Teil der Website inhaltlich zu verwalten.
